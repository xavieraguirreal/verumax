# Estrategia Comercial y Tarifas - Certificatum

**Versión:** 2.1
**Última actualización:** Enero 2026

---

## Resumen Ejecutivo

La estrategia de precios de Certificatum está diseñada para:

- Posicionar la solución como producto de **alta gama institucional**.
- Garantizar que los **afiliados** siempre tengan la mejor oferta del mercado (60% OFF).
- Proteger un **ingreso neto mínimo de $9.00 USD** en el plan base tras comisiones.
- Incentivar la **suscripción anual** y el compromiso mediante el costo de **Alta Bonificada**.

## 5. Programa "Founder Status" (Descuento Vitalicio)

Política de **Descuento Vitalicio Porcentual** para premiar a los usuarios iniciales. El Founder mantiene el descuento con el que se suscribió originalmente.

### Tipos de Founder

| Tipo                 | Canal de compra                   | Descuento vitalicio |
| -------------------- | --------------------------------- | -------------------:|
| **Founder Web**      | Suscripción directa en web        | **50% OFF**         |
| **Founder Afiliado** | Suscripción con cupón de afiliado | **60% OFF**         |

### Condiciones Generales

| Regla            | Descripción                                                              |
| ---------------- | ------------------------------------------------------------------------ |
| **Elegibilidad** | Primeros **500 clientes** o hasta **31/03/2026** (lo que ocurra primero) |
| **Beneficio**    | Mantiene el % de descuento con el que se suscribió (50% o 60%)           |
| **Protección**   | Si el PVP sube, el Founder mantiene su % sobre el nuevo valor            |
| **Pérdida**      | Si cancela, pierde el estatus y vuelve con tarifa vigente                |

### Ejemplo de Protección contra Inflación

```
Escenario: Supremus sube de $200 a $250 (nuevas funciones Blockchain)

Usuario Nuevo (Web):
→ Paga: $250 × 0.50 = $125.00/mes

Usuario Nuevo (Afiliado):
→ Paga: $250 × 0.40 = $100.00/mes

Founder Web (50% OFF vitalicio):
→ Paga: $250 × 0.50 = $125.00/mes (mismo % que al inicio)

Founder Afiliado (60% OFF vitalicio):
→ Paga: $250 × 0.40 = $100.00/mes (mismo % que al inicio)
```

**Nota:** En este ejemplo el precio no cambia porque el % se mantiene. La protección real se activa si en el futuro se reduce el descuento estándar (ej: Web baja de 50% a 40% OFF).

## Notas de Implementación

### Para Desarrollo

- Nombres en código: `singularis`, `essentialis`, `premium`, `excellens`, `supremus`
- Precios en centavos USD para evitar errores de redondeo
- El cupón aplica 60% OFF sobre `PVP_LISTA`
- Alta ($15) como ítem separado en primera factura
- Validar cupón contra tabla `afiliados_cupones`

### Para Marketing

| Elemento           | Implementación                                                   |
| ------------------ | ---------------------------------------------------------------- |
| **PVP Ancla**      | Mostrar tachado en landing y checkout                            |
| **Promo Web**      | Badge "50% OFF Lanzamiento" con contador (aunque sea permanente) |
| **Cupón Afiliado** | Solo visible en checkout al aplicar código, nunca público        |
| **Founder Badge**  | Mostrar en dashboard del usuario fundador                        |
| **Urgencia**       | "Quedan X lugares de 500" o countdown a 31/03/2026               |

### Para Ventas

- El Plan Singularis ($2/cert) es herramienta de comparación, no para promocionar activamente
- Siempre ofrecer anual primero (Alta gratis + mejor precio)
- Usar calculadora: "Con 50 certs/mes, Singularis = $100, Essentialis = $12"
